# 🚀 CRM Dashboard UI 
## 🏢 Overview
This project is a CRM Dashboard UI designed for a digital agency handling 30+ active clients.
The dashboard helps teams efficiently manage:
- Leads
- Clients
- Tasks
- Follow-ups
It focuses on clarity, usability, and real-world workflow structure.

## 🎯 Objective
To design a high-conversion, scalable SaaS dashboard UI that:
- Improves lead tracking visibility
- Reduces missed follow-ups
- Streamlines client management
- Supports daily operational workflows

## 👥 Target Users
- Digital Agencies
- Freelancers
- Small Business Teams

## 🔄 Lead Pipeline Flow
New → Contacted → Qualified → Converted
This ensures clear tracking from first interaction to final conversion.

## 🧩 Key Features
### 📊 KPI Cards
- Total Leads
- Active Clients
- Revenue
- Pending Tasks
### 📌 Lead Pipeline (Kanban)
- Stage-based lead tracking
- Organized workflow visibility
### 📋 Client Management
- Table-based structured client list
- Easy scanning and updates
### ✅ Task & Activity Tracking
- Follow-ups
- Daily tasks
- Status monitoring

## 🧠 Design Approach
- Clean and minimal UI
- Strong visual hierarchy
- Consistent spacing and layout
- Optimized for quick decision-making

## 🛠️ Tools Used
- Figma (UI Design & Prototyping)
