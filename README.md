# CRM Dashboard UI
## Overview
A web-based CRM dashboard designed for a **digital agency managing 30+ clients**.
It helps track:
Leads
Clients
Tasks
Follow-ups
## Objective
To design a **scalable SaaS dashboard UI** that improves:
Lead tracking
Client management
Workflow efficiency
## Workflow
Lead pipeline follows:
New → Contacted → Qualified → Converted
## Key Features
KPI Cards (Leads, Clients, Revenue, Tasks)
Lead Pipeline (Kanban-style)
Client List (table view)
Task & Activity tracking
